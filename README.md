# ✈️ TravelUp

Um jogo de conscientização sobre o tráfico de pessoas disfarçado de uma simples experiência de compras online.

## 🎮 Sobre o Jogo

Em **Viagem dos Sonhos**, você controla apenas o ponteiro do mouse enquanto utiliza um computador para realizar compras online.

Seu objetivo parece simples: comprar uma passagem aérea para assistir à Copa do Mundo de 2026. Porém, durante a navegação, diversos pop-ups, anúncios, promoções e ofertas aparecem na tela.

Algumas ofertas são legítimas, mas outras escondem golpes e situações inspiradas em casos reais de tráfico de pessoas e exploração humana.

O desafio é identificar quais oportunidades são seguras, fechar conteúdos suspeitos e concluir sua compra sem cair em armadilhas.

## 🎯 Objetivo

- Comprar uma passagem aérea para a Copa do Mundo de 2026.
- Identificar ofertas confiáveis.
- Fechar anúncios e mensagens suspeitas.
- Evitar golpes e propostas enganosas.
- Aprender a reconhecer sinais de tráfico de pessoas e exploração.

## 🖱️ Controles

| Ação | Controle |
|--------|---------|
| Mover cursor | Mouse |
| Interagir | Clique esquerdo |
| Fechar janelas | Clique esquerdo |

O jogo é controlado exclusivamente pelo mouse.

## ⚠️ Temática

O projeto foi desenvolvido com foco na conscientização sobre:

- Tráfico de pessoas;
- Golpes de recrutamento;
- Promessas falsas de emprego;
- Exploração laboral;
- Manipulação através da internet;
- Segurança digital.

As situações apresentadas são inspiradas em estratégias utilizadas por criminosos para atrair vítimas, adaptadas para fins educativos.

## 🧠 Mensagem

Muitas vítimas de tráfico humano são atraídas por promessas aparentemente inofensivas, como empregos, viagens ou oportunidades financeiras.

Aprender a identificar sinais de alerta pode ajudar a prevenir situações de exploração e proteger pessoas vulneráveis.

## 🛠️ Tecnologias Utilizadas

- GameMaker Studio 2
- GML (GameMaker Language)

## 📚 Propósito Educacional

Este jogo foi criado como um projeto educacional para promover reflexão e conscientização sobre um problema real que afeta milhões de pessoas em todo o mundo.

## 👨‍💻 Desenvolvimento

Projeto desenvolvido por estudantes com o objetivo de utilizar jogos digitais como ferramenta de educação e impacto social.
